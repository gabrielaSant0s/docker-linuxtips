### Terminal
#### - Ver se quais containers estão em execução

###### nova

``` docker container ls ```

###### velha

``` docker ps```

#### - Executar container

``` docker container run -t <nome da imagem docker>```

#### - Mostra todos os containers que já foram executados na máquina

``` docker container ls -a```

### Teclado 

#### sair do container sem matá-lo
